System Toolkit

A lightweight utility suite written in Python with a Windows batch launcher.

Features

  • Clear temporary files
  • Display system information
  • Simple, clean menu interface

How to Run

  1. Install Python 3.8 or newer
  2. Download or clone this repository
  3. Double‑click run_toolkit.bat (Windows)
  4. Or run python toolkit.py manually

Requirements

No external libraries required.
